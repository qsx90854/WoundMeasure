使用者需求：
請寫一隻python程式, 有UI 可以讓我手動選取影片1和影片2 , 並將這兩個影片接在一起變成影片3存在指定目錄

解決方案與設計說明：
我們新建立了一隻名為 `video_merger.py` 的 Python UI 程式，設計要點如下：
1. **多重拼接模式支援**：
   - 「前後相接 (Concatenate)」：順序將影片 1 播放完後播放影片 2。若解析度不同會自動以影片 1 為基準進行 resize 以防寫入失敗。
   - 「左右拼接 (Side-by-Side)」：將兩部影片的幀水平拼貼在一起。高度會自動對齊影片 1。
   - 「上下拼接 (Stacked)」：將兩部影片的幀垂直拼貼在一起。寬度會自動對齊影片 1。
2. **圖形化使用者介面 (UI)**：
   - 使用 Python 內建的 `tkinter` 與 `ttk`，提供原生流暢的跨平台界面。
   - 包含對應的「瀏覽...」與「選擇路徑...」按鈕，方便手動選取輸入與輸出檔案。
3. **後台背景處理 (Threading)**：
   - 核心拼接邏輯使用背景線程（Threading）執行，不會因為長時間的影片轉檔導致 Tkinter UI 視窗凍結或顯示「沒有回應」。
4. **進度提示與防錯處理**：
   - 界面自帶實時進度條 (Progressbar) 和處理幀數資訊的提示。
   - 在轉檔結束時，主動彈出確認視窗，提升使用者交互體驗。
