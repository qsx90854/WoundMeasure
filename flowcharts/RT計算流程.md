# RT 計算流程（2026-07-07 版）

實作位置：`Algorithm/video_pose_analysis.py` → `analyze_video_frames()`
適用：`depth_measure_multi_aruco_sbs_camera_v7_demo_zebra.py`（及任何 import 此模組的主程式）

## 模式

- **少標籤模式**（共享標籤 ≤ 2，主要使用情境）：ref 標籤（最小 ID）IPPE 雙分支 + 特徵極線消歧。
  標籤**不假設共面**；非 ref 標籤只用於尺度估計與極線驗證。
- **多標籤模式**（≥3）：原標籤地圖 + Joint PnP 路徑（未改動，無分支保護）。

## 流程圖

```
載入影格 → 前半段(右圖候選)/後半段(左圖候選) → 漸進抽樣(10→50幀/段)
  → 每幀: CLAHE → ArUco偵測 → cornerSubPix
  → 共享標籤 ≤2 ?
      是 → ref標籤 solvePnPGeneric(IPPE_SQUARE) 保留 2 分支
      否 → 標籤地圖平均 + Joint PnP 單解
  → 配對枚舉: (s,e) × 分支組合, R_rel=Rs·Reᵀ, t_rel=ts−R_rel·te
      閘門: baseline 8~220mm
      初分數: ref標籤自我擬合殘差 + baseline偏離45mm/模糊/覆蓋率/標籤數懲罰
  → 特徵極線重排(前16名): SIFT匹配(ratio 0.75+mutual, 去畸變) 
      中位對稱極線距離 ×0.5 權重加入分數 → 分支錯的組合被淘汰
  → 收斂判定: 少標籤看 feat_epi < 0.8px (多標籤看 reproj < 0.2px)
      未過 → 擴大抽樣進下一階段
  → 選定最佳對(記錄分支) + 次佳對 ≤5 組(feat_epi < 1.5px, 每起始幀取最佳分支)
  → 混合 RT 精修(每一對):
      SIFT → findEssentialMat(RANSAC 0.75px) → recoverPose → R_E, t方向
      閘門: 內點≥25; 旋轉差≤10° 或與另一IPPE分支一致(判定ArUco翻面, 放行)
      尺度: (R_E, 單位t) 三角化所有共享標籤角點 → 邊長中位數 vs 8.25mm → baseline
      仲裁: 特徵極線殘差需改善 且 標籤角點極線殘差退步 ≤0.3px
      → 通過: R,t方向=特徵解, 尺度=邊長定標 / 不通過: 保留 ArUco 解
  → 輸出:
      參考平面 = ref標籤角點以最終RT三角化 → SVD擬合 (與量測同幾何鏈, 誤差相消)
      HUD姿態指標 = 最終RT的特徵極線殘差 (取代無鑑別力的自我擬合殘差)
      R_rel / t_rel / baseline / 最佳+次佳影格 / SIFT特徵 → 主程式
```

## 設計原理

1. **IPPE 雙分支**：小平面標籤 PnP 有兩個殘差幾乎相同的鏡像解（實測 7e-6 vs 0.016px），
   殘差選不出來，必須保留雙解交給獨立證據裁決。
2. **特徵極線裁決**：標籤角點對分支無鑑別力（任何分支組合都完美重投影回自己）；
   場景 SIFT 點是唯一獨立證據，分支錯時極線殘差大一個數量級。
3. **信任分工**：R/t 方向 ← 特徵 E-matrix（幾百點）；尺度 ← 標籤已知邊長（8.25mm）；
   標籤角點 ← 仲裁第二證人。非 ref 標籤僅貢獻尺度樣本與驗證點（不需共面）。
4. **平面與點同鏈**：參考平面用量測同一組 RT 三角化擬合，系統誤差在高度相減時抵消。

## 關鍵 log 對照

| log | 意義 |
|---|---|
| `ℹ️ [少標籤模式] ...` | 進入分支感知路徑 |
| `🎯 [pair quality] ... feat_epi=...` | 候選對品質（feat_epi 是真幾何指標）|
| `🎉 ... 特徵極線殘差 X px (IPPE 分支 (s,e))` | 配對與分支裁決結果 |
| `🔀 [RT精修] ...判定 ArUco 落入錯誤分支` | 偵測到翻面，由特徵解接手 |
| `📏 標籤已知邊長定尺度: baseline A → B mm` | baseline 來源（應與實際移動量級相符）|
| `🚀 採用混合解 / ℹ️ 保留 ArUco RT` | 最終 RT 來源 |
| `📐 [三角化平面] ... RMS 殘差` | 平面品質（正常 <1mm）|
| `✅ [參考平面] 由 ref 標籤 ID:X ...` | 傷口高度的基準面 |

## 已知限制

- 多幀光流模式（去漂移/純光流）的逐幀 pose 仍為各幀分支 0，未經消歧。
- ≥3 標籤走舊路徑，平面為多標籤合併（不規則表面上不適用）。
- 場景需有足夠紋理供 SIFT 匹配；純平坦無紋理場景下分支消歧與混合精修會退化為保留 ArUco 解。
